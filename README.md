🛡️ AI Phishing Website Detection (Chrome Extension)

🚀 Overview

This project is a Chrome Extension that detects phishing websites using AI techniques and alerts users in real-time to prevent cyber threats and malicious attacks.

🔥 Features

* 🔍 Real-time phishing website detection
* 🤖 AI/ML-based classification
* ⚡ Fast and lightweight Chrome extension
* 🚨 Instant alerts for unsafe websites
* 🎯 User-friendly interface

🧠 Tech Stack

* JavaScript (Chrome Extension APIs)
* HTML, CSS
* Machine Learning (for phishing detection)

📂 Project Structure

ai-phishing-detection-extension/
├── manifest.json        # Extension configuration
├── background.js        # Background/service worker
├── content.js           # Website monitoring
├── popup.html           # Extension UI
├── popup.js             # UI logic
├── styles.css           # Styling
├── icons/               # Extension icons
└── README.md

⚙️ Installation (Run Locally)

1. Open Chrome and go to:
    chrome://extensions/
2. Enable Developer Mode (top right)
3. Click Load Unpacked
4. Select this project folder


🎯 How It Works

* The extension monitors the current website URL
* The URL is analyzed using an AI/ML model
* The system classifies the site as Safe or Phishing
* If phishing is detected, a warning alert is shown to the user
